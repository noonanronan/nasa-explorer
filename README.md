# NASA Explorer
A full-stack web app built for the Bounce Insights. The app fetches and displays space related data from NASA's public APIs.

## Technologies
● React
● NodeJS
● Express
● NASA Open APIs
● Axios
● CORS
● Dotenv

## Structure
■ ├── frontend/
■ ├── backend/
■ └── README.md

## Setup Instructions

Backend setup
```bash
cd backend
npm install
# Add your NASA_API_KEY to .env
node index.js
```
